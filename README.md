# prompt-tasks-writer-vue

NovelAI绘图任务队列生成器

An tampermonkey script written in Vue 3, used to automate custom drawing tasks on novelai.net/image.

一个使用vue3编写的油猴插件，用于在novelai.net/image自动化执行自定义的绘图任务。
